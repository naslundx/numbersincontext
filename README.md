# Numbers in context

## Install

`pip install poetry`
`poetry install`

## Test

For testing

`npm test`
`poetry run black .`
`poetry run pytest .`

For linting

`python -m pylint .`

For setting up basic DB

`python3 -m backend.setup`

## Build

Rebuild website:

`npm run build`

For backend:

`python server.py`
